kitchen sink

Kitchen Sink embodies my way of imagining personal sites: messy but a bit of everything. I’ve already written a number of articles around my blog engine, the motivation behind it and some infos. The most important article, containing the architecture of Kitchen Sink is the original one.

Kitchen Sink is not a build tool although you could bend it a bit. We plan do to add some dependency-checking but that’s about it.

features

  • live-reload or one-time builtins
  • a few pre-built layouts (customizable as a library)
  • categories using tags
  • article summaries
  • commonmark as main input articles
  • tunable CSS, JS per page
  • microscriptable in Dhall
  • open-graph and twitter-card metas
  • dev-server
  • advanced content analysis features
  • rather fast considering all it does
